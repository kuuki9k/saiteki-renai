========================================
  シナリオ編集ガイド
========================================

■ フォルダ構成
  第0章/prologue.json     → プロローグ「最適の朝」
  第1章/common.json       → 共通ルート「接触の季節」
  玲央ルート/scenes.json  → 一条玲央ルート
  湊ルート/scenes.json    → 神崎湊ルート
  朔ルート/scenes.json    → 黒瀬朔ルート
  悠真ルート/scenes.json  → 橘悠真ルート

----------------------------------------
■ text フィールドの書き方
----------------------------------------
台詞・地の文は text フィールドを配列で書きます。
1要素 = ゲーム画面の1行です。

  "text": [
    "18歳の誕生日。",
    "今日が、すべての始まりだと、誰もが言う。"
  ]

空行（間をあけたいとき）は "" を入れます。

  "text": [
    "国家AIシステム「FATES」——",
    "",
    "今年から、18歳全員が対象になった。"
  ]

----------------------------------------
■ speaker フィールド（話者名）
----------------------------------------
  "speaker": "一条 玲央"   → キャラクターの名前を表示
  "speaker": null          → ナレーション（話者名なし）

----------------------------------------
■ 使えるプレースホルダー
----------------------------------------
  {playerName}      → プレイヤーが入力した「名前」（例：花子）
  {playerLastName}  → プレイヤーが入力した「苗字」（例：山田）
  {playerFullName}  → 苗字 + 名前（例：山田 花子）

  例）"「{playerLastName}さん、ですね」"
      → ゲーム中は「山田さん、ですね」と表示されます

----------------------------------------
■ 注意事項
----------------------------------------
・保存後にブラウザをリロードすると反映されます
・セリフ内の「"」はそのまま書かず \" と書いてください
・[ ] の中の要素はカンマ , で区切ります（最後の要素にはカンマ不要）
・ファイルを壊してしまったら、バックアップから復元するか
  Claude Code に修正を依頼してください
